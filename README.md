📝 Descrição do programa:



O projeto foi desenvolvido com o objetivo de colocar em prática os conhecimentos básicos de JavaScript utilizando o ambiente de execução Node.js. Durante seu desenvolvimento, foram aplicados conceitos essenciais para a construção de aplicações back-end, como modularização do código, funções assíncronas (async/await), manipulação de pacotes externos e interação com o usuário por meio do terminal.

Além disso, o projeto utiliza variáveis de ambiente para armazenar configurações da aplicação de forma mais segura e organizada, seguindo boas práticas de desenvolvimento.

Desenvolvido durante meus estudos de Node.js, o programa permite que o usuário interaja com a aplicação através do terminal, escolhendo entre duas funcionalidades principais: gerar um QR Code a partir da URL de um site ou criar uma senha aleatória. Ao longo do desenvolvimento, foi possível consolidar conhecimentos sobre a estrutura de projetos em Node.js, organização em módulos, gerenciamento de dependências com npm, utilização de bibliotecas externas e tratamento de operações assíncronas, proporcionando uma compreensão mais ampla do funcionamento de aplicações executadas fora do navegador.
